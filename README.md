# Retail Customer Analytics & RFM Segmentation

## Project Overview

This project analyzes retail customer transaction data to understand customer behavior, identify top customers, analyze sales trends, and perform customer segmentation using the RFM (Recency, Frequency, Monetary) model.

The project demonstrates data cleaning, exploratory data analysis (EDA), visualization, customer analytics, and business insight generation using Python.

---

## Objectives

- Analyze customer transaction behavior
- Identify top revenue-generating customers
- Perform monthly sales trend analysis
- Detect outliers in transaction data
- Segment customers using RFM Analysis
- Analyze customer churn and response patterns

---

## Dataset

The project uses two datasets:

### Retail_Data_Transactions.csv
Contains:
- Customer ID
- Transaction Date
- Transaction Amount

### Retail_Data_Response.csv
Contains:
- Customer ID
- Customer Response
